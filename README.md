Design and develop a modern, minimal, developer-focused website for a startup called **Blackstart Labs**. The brand philosophy is: *Great systems begin in silence and awaken with purpose.* The site should feel like it was built by engineers, not marketers.

---

**Pages Required**

- Home
- Services
- Courses
- Open Source
- About
- Contact

---

**Home Page Structure**

Hero section with the company name, tagline, and a single CTA button. No stock photos, no illustrations, no carousels. Just typography, whitespace, and intent. Below the hero — a brief what we do section, featured services, featured courses, open source projects, and a closing CTA.

---

**Design Direction**

- Dark theme as default — deep black or dark navy background
- Accent color — electric blue or sharp cyan, used sparingly
- Typography — monospace or geometric sans-serif for headings, clean readable font for body
- No gradients, no glassmorphism, no excessive animations
- Minimal UI — every element earns its place
- Feels like a serious engineering company, not a bootcamp or agency template

---

**Services Section**

Cover these offerings clearly:
- Full-stack web development (ASP.NET Core, Next.js, React)
- Backend engineering (C#)
- AI integration and automation solutions
- API design and development

---

**Courses Section**

Grid or card layout showing available courses. Each card should have a title, short description, tech stack tags, and a CTA. Topics include full-stack development, competitive programming, backend engineering, and AI tools for developers.

---

**Open Source Section**

Showcase GitHub projects with name, description, tech stack, star count, and a link to the repo. Pull data from GitHub API if possible.

---

**About Section**

Short, honest, no fluff. Who built this, why it exists, what the mission is. No "we are passionate about technology" filler.

---

**Tech Stack for Development**

- Frontend — Next.js 14 with TypeScript
- Styling — Tailwind CSS
- Animations — Framer Motion, minimal usage
- CMS — Notion API or MDX for courses and blog
- Deployment — Vercel

---

**Tone & Copy Direction**

- Sharp and direct
- No buzzwords, no hype
- Sounds like a developer wrote it, not a marketing team
- Short sentences, confident statements

---

**What to Avoid**

- No stock photography
- No gradient hero backgrounds
- No excessive micro-animations
- No generic agency template feel
- No light mode as default
